The schedule app is made as a simple Spring Boot application to make a rest api.
Made using Java 17, with maven or mvnd.
With a demo made in bash.
To run bash demo run in a bash shell,
```
bash demo-api.sh 8081
```
This will run a script that builds the project and starts it on port 8081, and executes request against the api.
to run the unit test use
mvn test or mvnd test
